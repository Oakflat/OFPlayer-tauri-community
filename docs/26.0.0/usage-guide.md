# OFPlayer Tauri v26.0.0 使用指南

## 首次启动

### 引导流程

首次打开 OFPlayer 会显示引导页面，介绍核心功能。引导完成后不再自动显示，可通过设置重新打开。

### 隐私诊断同意

引导关闭后，如果尚未选择，会弹出隐私诊断同意对话框：
- **接受**：允许收集匿名诊断数据（启动性能、错误信息）
- **拒绝**：不收集任何诊断数据

此设置可在设置中随时更改。

## 曲库管理

### 导入音频

1. 点击左侧导航栏底部的「导入」按钮
2. 选择音频文件或包含音频的文件夹
3. OFPlayer 会将音频复制到托管存储目录
4. 自动解析元数据（标题、艺术家、专辑、封面等）

### 托管存储

OFPlayer 将导入的音频复制到用户选择的存储目录：

```
<storageRoot>/OFPlayer Library/libraries/<libraryId>/
```

设置中可查看存储占用详情。

### 扫描目录

在设置中配置扫描目录后：
- **手动扫描**：点击「扫描」按钮
- **自动扫描**：开启「启动时自动扫描」，应用启动时自动扫描

### 多曲库

支持创建多个曲库，每个曲库独立管理：
- 左侧面板切换曲库
- 每个曲库可有自己的歌单
- 右键曲库可重命名或删除

## 播放

### 基本操作

- **播放/暂停**：底部播放栏中央按钮
- **上一首/下一首**：播放栏两侧按钮
- **拖动进度**：点击或拖动进度条
- **音量调节**：播放栏右侧音量滑块

### 播放队列

- 点击曲目列表中的任意曲目开始播放
- 自动将当前曲库剩余曲目加入队列
- 支持歌单内连续播放

### 输出设备

在设置中可选择音频输出设备：
- 自动使用系统默认设备
- 手动选择特定设备
- 支持切换时无缝过渡

### 系统媒体控制

Windows 下支持：
- 任务栏缩略图媒体控制
- 锁屏媒体控制
- 键盘媒体键

## 歌词

### 歌词匹配

OFPlayer 按以下优先级匹配歌词：
1. 曲目同目录下的同名 `.lrc` 文件
2. 设置中配置的额外歌词目录
3. 文件名匹配（不区分扩展名）

### 歌词显示

- 主面板右侧歌词面板显示当前歌词
- 时间戳歌词会自动高亮当前行
- 支持点击歌词行跳转

### 歌词胶囊（Lyric Capsule）

歌词胶囊是一个独立的轻量 HUD 窗口：

- **打开/关闭**：播放面板右上角胶囊按钮
- **功能**：显示当前歌词行、播放进度、音浪
- **特性**：
  - 始终置顶
  - 透明背景
  - 无边框
  - 不在任务栏显示
  - 本地进度插值（不依赖主窗口高频推送）

## 沉浸式播放器

点击播放面板顶部的全屏按钮进入沉浸模式：
- WebGL 渲染的专辑色彩背景
- 歌词居中显示
- 底部播放控制栏
- 玻璃模糊效果

## 外部曲库

支持连接远程音乐服务：

1. 打开外部库对话框
2. 输入 WebDAV 或 Navidrome 服务器地址
3. 输入认证信息
4. 浏览和播放远程曲目

远程曲目支持播放缓存，已播放的曲目会缓存到本地。

## 设置中心

### 外观

- 语言：中文 / English
- 主题：Mist / Paper / Material
- 颜色方案：浅色 / 深色 / 跟随系统
- 动效：开/关
- 窗口效果：无 / 轻微 / 全部

### 播放

- 输出设备选择
- 启动时记住音量

### 曲库

- 存储目录配置
- 扫描目录管理
- 歌词目录管理
- 自动扫描开关
- 存储占用分析
- 存储清理（GC）
- 清空全部数据

### 其他

- 技术元数据显示开关
- 隐私诊断同意管理
- 重新打开引导
