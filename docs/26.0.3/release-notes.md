# OFPlayer v26.0.3 Release Notes

日期：2026-05-27
类型：桌面体验与诊断上传修复迭代

## 版本报文

改善了播放器的性能，取消外圈毛玻璃独立样式，目前以更优秀的桌面形态呈现，更新了封面功能。

## 主要改进

- 改善播放器性能与启动后的交互响应。
- 取消外圈毛玻璃独立样式，让桌面窗口以更统一、更稳定的形态呈现。
- 更新封面功能，提升曲库封面显示与可见曲目封面补全体验。
- 修复诊断日志上传能力：没有显式构建环境变量时默认使用生产诊断接口，上传前会等待本地日志落盘，并保留用户同意与主动上传门槛。

## 验证重点

- 社区分支保留本地诊断日志，不包含官方诊断上报端点。
- 本地诊断日志在上传前会刷新到磁盘。
- 缺失或空诊断日志仍会生成合法的脱敏上传报告。
- 构建产物需要内置生产 API、遥测与诊断地址。
